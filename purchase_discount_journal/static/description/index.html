Purchase Discount Journal On Invoice

This module allows you to mention discount Amount,Total of Customer/Supplier Invoice and Discount Journal Entry

1. Create Journal
Create Journal Entry Purchase. Give code 'SD' for sale journal and code 'PD' for purchase journal. System will search default journal for sale /Purchase.

2. As Fixed
Select 'Fixed' from Discount type and give discount Amount as Discount rate. System will update the value of Discount and Total

And the module also allows you to set a journal for discount Amount.
